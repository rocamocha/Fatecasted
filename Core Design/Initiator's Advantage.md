The player to go first during a round should have a clear advantage, so that offense feels powerful and defense is engaging. Possible ways to accomplish this:
- Certain options only available to the initiator.
- Higher potency of effects or triggers while initiating.

# What we already do.
Currently, the ways in which initiator's advantage is realized is through:
- Basic actions only available to the initiator.
- Being able to play Fates from their hand on the action phase.
- First choice out of the shared dice roll.
- Option to end the action phase early.
- Fate stack before the action phase.
