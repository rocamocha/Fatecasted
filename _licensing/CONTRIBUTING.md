# Contributing to Fatecasted

Thank you for your interest in contributing to Fatecasted! We're excited to build this universe together. To ensure a smooth and collaborative process, please review these guidelines.

## Code of Conduct

First and foremost, all contributors are expected to follow our [Code of Conduct](CODE_OF_CONDUCT.md). Please be respectful and constructive to foster a welcoming community for everyone.

## How to Contribute

We welcome contributions of all kinds, from fixing typos to proposing new game mechanics or submitting code for our tools.

1.  **Find an Issue or Propose an Idea:** You can start by looking at existing issues in the repository. If you have a new idea, please open a new issue to discuss it before you start working on it. This helps us stay coordinated.
2.  **Fork the Repository:** Create your own copy of the repository to work on.
3.  **Create a Branch:** Make a new branch in your fork for your changes.
4.  **Make Your Changes:** Edit the files and commit your work with clear, descriptive messages.
5.  **Submit a Pull Request:** Open a pull request from your branch to the `main` branch of the official Fatecasted repository. Please provide a clear description of the changes you've made.
